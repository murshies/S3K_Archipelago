This directory contain information about all of the pickups that are considered as part of this Archipelago. Each act of each zone has its own YAML file containing the list of pickups. Additionally, the file `types.yaml` contains type hierarchy information for each pickup type. This will allow the apworld to group pickups into broader categories. For example, the lightning, flame, and water shields have a super type "shield", which in turn has the super type "item_box".

There is also a map of each act, which is annotated with numbers to illustrate where each pickup is in the act. These maps were originally pulled from https://info.sonicretro.org/Sonic_the_Hedgehog_3_%26_Knuckles/Maps

The file `SUMMARY.md` contains a tabulated summary of the number of each pickup type, both for game totals and per zone.
